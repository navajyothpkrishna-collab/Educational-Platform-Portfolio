This project is a personal educational portfolio website developed using HTML, CSS, and JavaScript. It is designed to showcase academic details, skills, projects, and achievements in a clean, responsive, and interactive format.
The website acts as a digital resume, making it easy to present professional and educational information online.
Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout (Flexbox/Grid, animations)
JavaScript – Interactivity and form validation
Purpose
The main goal of this project is to:
Demonstrate front-end development skills
Apply UI/UX design principles
Create a professional online presence
